# Overlays

Kind: global class

# overlays.fetch(options) ⇒ Promise.<buffer>

Draws one image on a requested image overlay. Please see https://weebyapi.xyz/api/docs#overlays for the available overlays.

Kind: instance method of Overlays
Returns: Promise.<buffer> - The generated image in a buffer.

Param Type Description
options Object The options that contain the required parameters.
options.type string The Type of Overlay.
options.image string The link to an image (.png, .jpg, .gif)
