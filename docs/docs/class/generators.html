# generators.oneImage(options) ⇒ Promise.<buffer>

Draws one image on a requested image generator. Please see https://weebyapi.xyz/api/docs#generators for the available generators.

Kind: instance method of Generators
Returns: Promise.<buffer> - The generated image in a buffer.

Param Type Description
options Object The options that contain the required parameters.
options.type string The Type of Generator.
options.image string The link to an image (.png, .jpg, .gif)

# generators.spotifySearch(options) ⇒ Promise.<buffer>

Generates a Spotify Banner of statistics and information from either music, playlists or users. Please see https://weebyapi.xyz/api/docs#generators for the available generators.

Kind: instance method of Generators
Returns: Promise.<buffer> - The generated image in a buffer.

Param Type Description
options Object The options that contain the required parameters.
options.type 'album' | 'artist' | 'playlist' | 'track' | 'user' The type of Spotify Search. (album, artist, playlist, track or user)
options.query string The value to search up. (Use %20 for spaces)
